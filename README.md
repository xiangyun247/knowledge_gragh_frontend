# 医疗知识图谱系统 - 前端项目

> 基于 Vue.js 构建的现代化医疗知识图谱系统前端应用，提供智能问答、知识图谱可视化、实体搜索等核心功能。

## 📋 项目简介

医疗知识图谱系统是一个基于人工智能技术构建的医疗知识网络平台，将海量医疗数据结构化存储，通过图谱可视化方式展示医疗实体之间的关系。系统支持自然语言问答、实体搜索、知识图谱可视化、数据上传等功能，帮助用户快速获取和理解医疗知识。

## ✨ 核心特性

### 🎨 双主题系统
- **科技风（Tech Style）**：赛博青绿配色，霓虹发光效果，丰富的动画和粒子效果
- **医疗风（Medical Style）**：经典白蓝配色，简洁干净的界面，适合长时间使用
- 一键切换主题，主题选择自动保存

### 🚀 主要功能模块

#### 1. **封面页（Cover Page）**
- 系统介绍和欢迎信息
- 知识图谱概览数据展示
- 快速入口按钮（开始问答、查看图谱）
- 系统状态显示和进入系统按钮

#### 2. **首页（Home）**
- **系统能力**：6个功能卡片，两列三行布局
  - 智能问答
  - 图谱可视化
  - 实体搜索
  - 数据上传
  - 历史记录
  - 常见问题
- **系统监控**：实时数据统计展示
  - 总实体数
  - 关系总数
  - 搜索量
  - 用户活跃度
- **实时监控**：数据可视化图表
  - 实体类型分布（饼图）
  - 关系类型分布（柱状图）
  - 知识增长趋势（折线图）

#### 3. **智能问答（Chat）**
- 自然语言问答功能
- 实时对话界面
- 关联实体信息展示
- 示例问题快捷按钮
- 打字动画效果

#### 4. **知识图谱可视化（Graph）**
- 交互式图谱展示
- 实体和关系的可视化
- **图谱切换功能**：支持在下拉菜单中选择不同的知识图谱进行查看
- 缩放、拖拽、筛选功能
- 实体详情查看
- 图谱导出功能

#### 5. **实体搜索（Search）**
- 多类型实体搜索（疾病、症状、药物、治疗等）
- 模糊匹配和精确匹配
- 搜索结果卡片展示
- 实体详情查看

#### 6. **历史记录（History）**
- 完整的操作历史记录
- 按时间、类型筛选
- 搜索历史内容
- 批量删除功能

#### 7. **数据上传（Upload）**
- 支持多种文件格式（TXT、PDF、JSON）
- 拖拽上传
- 上传进度显示
- 数据解析结果展示

#### 8. **常见问题（FAQ）**
- 分类问题展示
- 搜索功能
- 相关问题推荐
- 反馈表单

#### 9. **用户中心（Profile）**
- 个人信息管理
- 头像上传和显示
- 表单验证
- 数据同步到 Vuex

- **Vue-particles 1.0.9** - 粒子效果组件

## 🚀 快速开始

### 环境要求
- Node.js >= 12.0.0
- npm >= 6.0.0 或 yarn >= 1.0.0

### 安装依赖

```bash
npm install
# 或
yarn install
```

### 开发运行

```bash
npm run serve
# 或
npm run dev
```

访问 http://localhost:8080

### 生产构建

```bash
npm run build
# 或
npm run build:prod
```

构建产物将输出到 `dist/` 目录

### 代码检查

```bash
npm run lint
```

## 🎨 主题系统

### 科技风（Tech Style）
- **配色方案**：赛博青绿 + 深空蓝
- **视觉效果**：霓虹发光、粒子动画、渐变背景
- **适用场景**：追求科技感和视觉冲击的用户

### 医疗风（Medical Style）
- **配色方案**：经典白蓝
- **视觉效果**：简洁干净、克制动画、清晰易读
- **适用场景**：需要长时间使用的医疗专业人员

### 主题切换
- 点击顶部导航栏右侧的主题切换按钮
- 主题选择自动保存到 localStorage
- 刷新页面后保持选择的主题

## 📱 响应式设计

系统已针对不同屏幕尺寸进行优化：

- **桌面端**（> 1024px）：三列布局，完整功能展示
- **平板端**（768px - 1024px）：自适应布局，优化间距
- **移动端**（< 768px）：单列布局，垂直排列

## 🔧 核心功能说明

### 1. 点击波纹效果
全局集成了 `v-ripple` 指令，为按钮和可交互元素添加点击波纹动画效果。

### 2. 滚动动画
使用 IntersectionObserver API 实现元素进入视口时的渐入动画。

### 3. 图表响应式
ECharts 图表会根据屏幕尺寸自动调整布局和图例位置。

### 4. 状态管理
使用 Vuex 统一管理用户信息、主题设置等全局状态。

### 5. 路由守卫
实现了路由级别的权限控制和布局控制（如封面页隐藏导航栏）。

## 📝 开发规范

### 代码风格
- 遵循 Vue.js 官方风格指南
- 使用 ESLint 进行代码检查
- 组件命名采用 PascalCase

### 组件开发
- 使用单文件组件（.vue）

### 样式规范

- 主题相关样式放在对应的主题文件中

## 🔍 常见问题

### Q: 如何切换主题？
A: 点击顶部导航栏右侧的主题切换按钮即可切换。

### Q: 如何上传头像？
A: 进入用户中心，点击头像区域，选择图片文件即可上传。

### Q: 图表不显示怎么办？
A: 检查浏览器控制台是否有错误，确保 ECharts 正确加载。

### Q: 移动端显示异常？
A: 清除浏览器缓存，或检查是否使用了响应式断点。
